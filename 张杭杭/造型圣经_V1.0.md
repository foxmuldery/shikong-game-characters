# 《失控游戏》张杭杭角色造型圣经 V1.0

## 一、角色定位

角色：张杭杭

年龄：23岁左右

身份：生活方式美妆博主

关系：邹青前女友

功能：团队中的观察者、记录者、信息读取者

核心关键词：

精致、敏感、观察、整理、记录、生活感。

一句话视觉定义：

> 一个2026年的城市生活方式女孩，外表干净精致，习惯记录生活细节，但进入失控空间后逐渐失去“体面外壳”。

---

# 二、人物视觉核心

张杭杭不是传统意义上的漂亮女孩。

她的核心不是展示自己，而是：

她习惯把生活整理成一个可控、可记录、可以发布的版本。

进入星星岛后，她第一次面对无法整理、无法预测、无法控制的世界。

人物冲突：

整理世界的人，进入一个无法整理的世界。

---

# 十一、人物核心变化

张杭杭最大的变化：

不是弱女子变强。

一个一直试图整理世界的女孩，最终接受世界本来就是混乱的。

“我要把一切整理好。”

“即使混乱，我也可以继续前进。”
