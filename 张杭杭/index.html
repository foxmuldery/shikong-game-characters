角色定位

张杭杭，23岁，生活方式美妆博主，邹青前女友。功能：观察者、记录者、信息读取者。核心冲突：整理世界的人，进入一个无法整理的世界。

核心视觉命题

一个2026年的城市生活方式女孩，外表干净精致，习惯记录生活细节，但进入失控空间后逐渐失去"体面外壳"。

脸部与发型

  • 清秀、安静、有观察力，避免网红脸/明星化
  • 黑色中长发，高位自然扎发，发丝略松
  • S0→S3：扎发完整 → 碎发增加 → 发尾散开汗湿 → 半散发混乱

基础定妆（A3 精致博主版）

部位描述
外套奶油粉短款轻夹克 — 翻领、双胸袋、金属按扣、柔软微宽松
内搭象牙白无袖针织背心 — 罗纹针织、贴身不性感
下装奶油白高腰直筒裤 — 干净宽松，禁止紧身/工装裤
鞋奶油白复古运动鞋 — 日常耐穿，禁止潮鞋

随身物资产

  • 奶油色小斜挎包（补妆纸、镜子、创可贴、湿巾）
  • 手机稳定器（记录世界，非专业摄影师）
  • 黑色发圈（腕上 → 紧张时取下 → 后期绑东西）
  • 小银色耳环（低调识别）

颜色系统

主色奶油粉 — 温柔、生活感、表面稳定
辅色象牙白 — 干净、整理、理想状态
变化不是换装，而是污染：粉变灰 → 灰尘污迹水渍 → 颜色仍在（她没有失去自己）

进入星星岛前后

阶段变化
BEFORE衣服干净、发型完整、包整理好、颜色明亮 — 观察别人
外套完整 → 打开 → 脱下 → 绑腰
发型完整 → 松散 → 湿乱
衣服干净 → 灰尘 → 污渍 → 磨损
行为记录别人 → 帮助别人 → 保护别人

剪影识别

高扎黑色马尾 + 短款粉色外套 + 粉白色系统 + 记录动作

人物核心变化

一个一直试图整理世界的女孩，最终接受世界本来就是混乱的。从「我要把一切整理好」变成「即使混乱，我也可以继续前进」。
