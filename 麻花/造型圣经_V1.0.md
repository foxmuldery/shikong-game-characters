# 《失控游戏》

# 麻花（MH-08）角色造型圣经 V1.0

## CHARACTER DESIGN BIBLE

版本：V1.0\
角色编号：MH-08\
角色名称：麻花 / Mahua\
类型：儿童记忆残留体 / 星星岛异常空间核心角色

------------------------------------------------------------------------

# 一、角色定位

## 一句话定义

> 一个2001年中国县城儿童乐园里没有离开的孩子。

麻花不是鬼童。

不是NPC。

不是游戏引导员。

她是晓晓没有被系统删除掉的真实记忆残留。

她代表：

-   等待；
-   不完整；
-   没有被听完的话；
-   没有被修补的真实。

------------------------------------------------------------------------

# 二、核心视觉命题

## 「被遗忘但仍然认真生活的孩子」

她不是恐怖来源。

恐怖来自：

这个孩子为什么还在这里？

为什么她记得所有路线？

为什么整个空间想删除她？

------------------------------------------------------------------------

# 三、造型三层逻辑

## 第一层：现实孩子

第一次出现：

观众应该认为：

"这是一个走失的小女孩。"

------------------------------------------------------------------------

## 第二层：儿童乐园工作人员残留

观众发现：

她不是普通游客。

她熟悉：

-   小火车路线；
-   员工通道；
-   维修门；
-   旧设备。

------------------------------------------------------------------------

## 第三层：记忆错误数据

真相：

她是晓晓没有被南建林修补掉的部分。

她保留：

-   哭；
-   等待；
-   生气；
-   不漂亮的瞬间。

------------------------------------------------------------------------

# 四、整体视觉关键词

## 关键词

-   中国2000年前后儿童乐园
-   县城商业空间
-   旧工作人员服
-   千禧儿童记忆
-   VHS录像感
-   褪色
-   磨损
-   未完成

------------------------------------------------------------------------

# 五、轮廓设计（Silhouette）

麻花必须做到：

> 不看脸，只看影子也能认出。

------------------------------------------------------------------------

## 三个核心轮廓符号

### 1. 双麻花辫

固定：

左： 暗红发绳

右： 旧蓝发绳

------------------------------------------------------------------------

### 2. 灰黄色工作马甲

必须明显大一号。

形成：

儿童身体 + 成人工作服的不协调感。

------------------------------------------------------------------------

### 3. 小孩比例

特点：

-   肩窄；
-   手臂细；
-   腿短；
-   身体轻。

------------------------------------------------------------------------

# 六、发型设计

## 双麻花辫

不是漂亮儿童发型。

而是：

一天工作结束后的状态。

------------------------------------------------------------------------

细节：

-   辫子略松；
-   发尾散开；
-   碎发明显；
-   一边稍低；
-   有灰尘感。

------------------------------------------------------------------------

## 发绳

红蓝不是鲜艳。

而是：

被时间洗掉颜色。

红：

暗红。

蓝：

旧蓝。

------------------------------------------------------------------------

# 七、面部设计

## 核心原则

第一眼：

普通。

第二眼：

异常。

第三眼：

心疼。

------------------------------------------------------------------------

禁止：

-   鬼妆；
-   白脸；
-   黑眼圈；
-   木偶笑；
-   恐怖儿童脸。

------------------------------------------------------------------------

## 表情

默认：

安静。

不是冷漠。

而是：

像一直等待某个人回来。

------------------------------------------------------------------------

## 眼神

特点：

-   观察；
-   熟悉；
-   有记忆。

她知道很多。

但她不知道如何解释。

------------------------------------------------------------------------

# 八、服装设计

## 1. 儿童乐园工作马甲

最重要视觉资产。

------------------------------------------------------------------------

材质：

旧尼龙。

------------------------------------------------------------------------

褪色灰黄。

------------------------------------------------------------------------

细节：

-   拉链氧化；
-   边缘磨白；
-   缝线脱落；
-   胸牌残胶；
-   星星岛贴纸残留。

------------------------------------------------------------------------

## 2. 内搭

旧灰白T恤。

要求：

普通。

没有潮流感。

------------------------------------------------------------------------

## 3. 下装

灰蓝儿童运动裤。

状态：

-   膝盖磨损；
-   裤脚旧；
-   洗旧泛白。

------------------------------------------------------------------------

## 4. 鞋

旧白布鞋。

细节：

-   鞋头发黄；
-   鞋底磨损；
-   鞋带旧。

------------------------------------------------------------------------

# 十一、正常状态造型

## EP01-EP08

状态：

完整。

------------------------------------------------------------------------

视觉：

-   颜色尚存在；
-   衣服完整；
-   辫子清楚；
-   表情自然。

------------------------------------------------------------------------

"她只是一个奇怪的小女孩。"

------------------------------------------------------------------------

# 十二、删除状态设计

## 第一阶段：颜色消失

变化：

-   发绳褪色；
-   马甲变灰；
-   贴纸颜色降低。

------------------------------------------------------------------------

## 第二阶段：影像损坏

变化：

-   衣服纹理缺失；
-   发丝边缘跳帧；
-   声音拖尾。

------------------------------------------------------------------------

## 第三阶段：记忆删除

变化：

-   手指边缘丢帧；
-   辫梢消失；
-   身体局部像录像损坏。

------------------------------------------------------------------------

原则：

不是透明鬼魂。

而是：

被删除的录像片段。

------------------------------------------------------------------------

# 十三、动作设计

## 行走

特点：

熟悉空间。

不像逃跑。

------------------------------------------------------------------------

## 停止

核心动作：

听声音。

头微微偏向广播方向。

------------------------------------------------------------------------

## 找路

不是思考。

而是身体记得。

------------------------------------------------------------------------

## 害怕

不是尖叫。

而是：

-   抓衣角；
-   后退；
-   找熟悉位置。

------------------------------------------------------------------------

# 十四、摄影设计

多拍：

-   背影；
-   辫子；
-   马甲；

------------------------------------------------------------------------

## 中期

增加：

侧脸。

------------------------------------------------------------------------

## 真相阶段

正面特写。

因为越接近真相，观众越应该看清她。

------------------------------------------------------------------------

# 十八、最终角色定义

> 麻花是一个穿着褪色儿童乐园工作马甲、扎着一红一蓝两根麻花辫的小女孩。她不是怪物，也不是鬼魂，她只是一个被留在2001年星星岛里的孩子，仍然认真守着那个已经不存在的游乐园。

# 十九、角色核心关系

球童男孩：

坏掉的快乐。

没有被听完的快乐。

两者共同构成第一关主题：

> 儿童空间里，一个东西坏掉了，一个孩子留下来了。
