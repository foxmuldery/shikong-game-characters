# 《失控游戏》角色造型圣经

# 周凯（K.Ghost）Character Bible V1.0

## 一、角色定位

角色名称：周凯

论坛ID：K.Ghost

年龄：25岁左右

身份：密室论坛玩家、都市探险爱好者、业余影像记录者。

核心关键词：

> 装备比人专业的人。

周凯不是专业探险家，也不是户外高手。他是一个长期沉浸在密室论坛、恐怖游戏和都市传说社区中的普通年轻人，逐渐相信自己已经成为"经验丰富的探险大神"。

他的造型核心：

> 把网络世界里的英雄形象，穿到了现实生活里。

------------------------------------------------------------------------

## 二、视觉一句话定义

一个把论坛头像里的自己实现到现实中的胖宅探险者。

周凯的视觉必须同时具备：

-   一点专业感；
-   一点中二感；
-   一点廉价感；
-   一点可爱感。

------------------------------------------------------------------------

## 三、整体视觉公式

周凯 =

圆润身体\
+ 黑框眼镜\
+ 黑色论坛文化T恤\
+ 旧橙色摄影马甲\
+ 大量廉价探险装备\
+ 半专业摄影设备

最终形成：

"民间密室大神"的唯一视觉识别。

------------------------------------------------------------------------

## 四、身体轮廓系统

### 身材定位

固定：

-   偏胖；
-   圆润；
-   厚重；
-   普通男性身体。

禁止：

-   健身型；
-   户外硬汉型；
-   英雄型。

周凯应该像一个普通办公室男性，但穿上了一整套探险装备。

------------------------------------------------------------------------

## 五、头部造型系统

### 面部

固定：

-   圆脸；
-   黑框眼镜；
-   普通男性五官；
-   微胖；
-   轻微疲惫感。

不要美化，不要英雄化。

### 眼镜

黑色粗框方形眼镜。

作用：

-   游戏宅；
-   论坛用户；
-   分析型人格。

### 发型

短黑发。

状态：

-   自然；
-   稍乱；
-   无明显造型感。

------------------------------------------------------------------------

# 六、服装系统

## 1. 黑色"K哥不怕鬼"T恤

这是周凯人格核心资产。

它不是普通衣服，而是他的身份宣言。

来源：

-   2010年前后淘宝定制文化衫；
-   密室论坛活动服；
-   小众玩家社群周边。

固定内容：

-   K哥不怕鬼；
-   ESCAPE ROOM；
-   密室逃脱实战团；
-   卡通探险版周凯；
-   论坛ID：K.Ghost。

质感：

-   老旧印花；
-   轻微裂纹；
-   洗旧；
-   非新品。

禁止：

-   星星岛信息；
-   商业品牌；
-   未来剧情信息。

------------------------------------------------------------------------

## 2. 橙色多口袋摄影马甲

周凯最大视觉锚点。

来源：

-   摄影记者马甲；
-   户外摄影装备；
-   淘宝探险马甲。

颜色：

旧橙色。

不是：

-   荧光橙；
-   时尚工装橙。

材质：

-   尼龙；
-   旧防水布；
-   磨损痕迹。

结构：

-   大量口袋；
-   胸袋；
-   腰袋；
-   背部网眼区域。

作用：

形成周凯独特剪影。

------------------------------------------------------------------------

## 3. 黑色功能裤

不是军事战术裤。

而是：

普通人买来的"探险裤"。

特点：

-   黑色；
-   宽松；
-   多口袋；
-   有使用痕迹。

------------------------------------------------------------------------

## 4. 鞋

旧款运动鞋。

特点：

-   灰黑色；
-   厚底；
-   磨损。

体现：

准备走很多路，而不是战斗。

------------------------------------------------------------------------

# 七、装备系统

周凯装备不是装饰，而是人格延伸。

## 运动相机

象征：

记录、证明、留下证据。

## 手电

象征：

探索幻想。

## 对讲机

象征：

把自己想象成行动队。

## 腰包

象征：

装备核心。

## 多功能工具

象征：

万能解决方案幻想。

## 记录本

象征：

论坛攻略习惯。

------------------------------------------------------------------------

## 装备原则

所有装备必须像：

-   淘宝买来的；
-   论坛推荐的；
-   自己慢慢攒出来的。

禁止：

-   特种装备；
-   军事装备；
-   科幻设备。

------------------------------------------------------------------------

# 十、状态变化

## G0 初始状态

"装备大神"。

特点：

-   马甲完整；
-   设备齐全；
-   T恤清晰；
-   自信。

## G1 第一关开始

变化：

-   马甲沾灰；
-   相机划伤；
-   手电掉漆。

## G2 中段

变化：

-   装备损坏；
-   口袋打开；
-   东西散落。

象征：

专业幻想崩塌。

## G3 后期

装备不再重要。

留下：

-   黑T；
-   眼镜；
-   普通人的脸。

------------------------------------------------------------------------

# 十一、喜剧原则

周凯不是笑话。

笑点来自：

他的认真。

例如：

他说：

"这个机关我研究过。"

结果：

机关升级。

他说：

"专业玩家都有备用方案。"

结果：

备用电池没电。

但关键时刻：

他的准备真的有用。

# 十三、最终定义

周凯不是一个穿装备的胖子。

他是：

一个相信自己是密室大神的普通人，把论坛里的英雄幻想全部穿在身上。

直到真正危险来临，他才发现：

最有用的不是装备。

而是有人愿意回答他的求救。
