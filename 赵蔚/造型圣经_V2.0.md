# 一、角色定位

## 赵蔚 Zhao Wei

年龄：20岁

身份： - 恐怖片爱好者 - 细节观察者 - 赵斌妹妹 - 团队中的信息观察者

角色关键词：

> 安静、克制、敏感、记录、观察。

赵蔚不是侦探，也不是黑客。

她的能力不是破解规则，而是发现别人忽略的异常。

她代表：

"一个普通人，因为认真观察，而逐渐接近真相。"

------------------------------------------------------------------------

# 二、核心视觉命题

## 一句话定义

> 一个习惯站在人群边缘记录异常的女孩，穿着普通生活服装进入失控空间，最终成为第一个敢于面对真相的人。

------------------------------------------------------------------------

# 三、整体美学方向

## 关键词

-   暗红观察者
-   低存在感
-   高信息密度
-   旧纸质感
-   生活真实感
-   克制女性气质

------------------------------------------------------------------------

赵蔚不能设计成：

-   女侦探；
-   神秘女主；
-   黑客少女；
-   文艺电影女青年。

她必须像：

一个2026年真实存在的普通大学女生。

但是：

只看背影，也能认出她。

------------------------------------------------------------------------

# 四、人物识别系统

## 剪影识别

赵蔚的剪影由五个元素组成：

1.  长线条外层
2.  暗红色上半身
3.  宽松深色长裤
4.  圆框眼镜轮廓
5.  旧帆布斜挎包

------------------------------------------------------------------------

## 与其他女性区分

### 张杭杭

张杭杭： - 奶油粉 - 短外层 - 腰线明显 - 生活方式博主

赵蔚： - 暗红 - 长线条 - 收束 - 记录者

------------------------------------------------------------------------

### 麻花

麻花： - 儿童比例 - 旧黄马甲 - 辫子

赵蔚： - 成年女性比例 - 深色 - 纸本记录

------------------------------------------------------------------------

### 晓晓

晓晓： - 浅黄 - 蓝白格 - 红色小包

赵蔚： - 暗红 - 灰黑 - 旧帆布包

------------------------------------------------------------------------

# 五、基础定妆造型

## 外层

暗红色旧衬衫 / 薄夹克。

颜色：

-   酒红
-   铁锈红
-   暗红棕

禁止：

鲜红、少女粉、紫红礼服感。

------------------------------------------------------------------------

材质：

-   洗旧棉布
-   微磨毛织物
-   哑光表面

特点：

有使用痕迹，但不是破旧。

------------------------------------------------------------------------

## 内搭

旧纸白色基础T恤。

作用：

中和暗红。

体现：

普通生活感。

------------------------------------------------------------------------

## 下装

深灰直筒裤。

要求：

-   宽松
-   实用
-   可行动

禁止：

紧身裤。

禁止：

裙装。

------------------------------------------------------------------------

## 鞋

黑色帆布鞋。

特点：

-   白边泛黄
-   鞋底磨损
-   鞋带旧

不是潮鞋。

不是运动达人鞋。

------------------------------------------------------------------------

# 六、面部与发型圣经

## 面部

核心参考：

真实生活感。

特点：

-   清淡五官
-   无强妆
-   眼神观察感
-   情绪内收

------------------------------------------------------------------------

## 眼镜

圆框细金属眼镜。

作用：

视觉识别点。

变化：

进入空间后：

-   镜片出现灰尘
-   雨水
-   划痕
-   反光遮挡

------------------------------------------------------------------------

## 发型

基础：

低马尾 / 自然束发。

特点：

-   黑色自然发
-   碎发
-   不刻意造型

区别：

张杭杭：

精致整理。

赵蔚：

功能优先。

------------------------------------------------------------------------

# 七、随身物资产

## 1. 旧帆布包

核心识别物。

颜色：

米灰旧帆布。

特点：

-   使用痕迹
-   小补丁
-   旧扣子
-   轻微污渍

------------------------------------------------------------------------

## 2. 记录本

最重要道具。

不是漂亮手账。

而是：

调查记录本。

包含：

-   时间
-   地点
-   异常记录
-   手写标记
-   旧照片

------------------------------------------------------------------------

## 3. 手机

功能：

记录证据。

不是科技设备。

------------------------------------------------------------------------

## 4. 小手电

普通黑色小手电。

体现：

准备意识。

------------------------------------------------------------------------

## 5. 透明证物袋

用于：

-   旧照片
-   纸片
-   小物件

# 十一、人物变化逻辑

赵蔚的变化不是：

从普通女孩变英雄。

而是：

从：

"记录别人"

变成：

"主动面对"。

------------------------------------------------------------------------

造型变化：

开始：

衣服保护她。

结束：

她穿着同一套衣服保护别人。

# 十四、最终视觉总结

赵蔚不是最强的人。

她也不是最勇敢的人。

她只是：

一直认真看。

所以最后，她看见了别人没有看见的东西。

她的造型应该始终保持：

> 一个普通女孩，在一个越来越不普通的世界里，坚持记录真实。
